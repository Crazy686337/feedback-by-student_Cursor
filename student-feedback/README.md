# Student Feedback System

Monorepo with backend (Express + MongoDB) and frontend (React + Vite + Tailwind). Provides student feedback submission and admin dashboard with charts.

## Quick start

1) Backend

```
cd backend
cp .env.example .env # update if using MongoDB Atlas
npm install
npm run dev
```

2) Frontend

```
cd ../frontend
npm install
npm run dev
```

Open the app URL printed by Vite (typically `http://localhost:5173`). Backend defaults to `http://localhost:5000`.
